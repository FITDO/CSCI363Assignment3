This project was created as Assingment 3 for CSCI 363 or Principals of Software Design.

It is a calculator that takes user input then evaluates the input and returns an answer to the problem

This particular approach converts user input into postfix form then an array is used to iterate throught all the commands

User Input: 2 x 2 + 4

PostFix to an array: 2 2 * 4 +

Answer: 8
